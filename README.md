# Usage
<!-- usage -->
```sh-session
$ npm install -g hoi4transdic
$ hoi4transdic COMMAND
running command...
$ hoi4transdic (-v|--version|version)
hoi4transdic/0.0.0 linux-x64 node-v12.18.3
$ hoi4transdic --help [COMMAND]
USAGE
  $ hoi4transdic COMMAND
...
```

